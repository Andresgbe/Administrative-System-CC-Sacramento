# Administrative-System-CC-Sacramento
Carrizal Shopping Center Admin System — a full-stack web app built with Angular, Node.js, Express and PostgreSQL that manages store rentals, tenant contracts, payments, and financial reporting, with support for multi-currency operations (Bs, USD, EUR) and two access roles: admin and subadmin.

## 🚀 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Angular + TypeScript |
| Backend | Node.js + Express |
| Database | PostgreSQL (Neon.tech / Supabase) |
| Authentication | JWT + bcrypt |
| Frontend Deploy | Vercel / Netlify |
| Backend Deploy | Railway / Render |

---

## 📦 System Modules

- **Stores** — Tenant management, contracts, and status tracking
- **Payments** — Rent registration and transaction history
- **Expenses** — Shopping center operational costs
- **Exchange Rates** — BCV, USDT and EUR with full history
- **Calculator** — Real-time conversion between Bs, USD and EUR
- **Dashboard** — KPIs, charts and general overview
- **Reports** — Export to PDF and Excel

---

## 👥 Access Roles

| Role | Permissions |
|------|------------|
| `admin` | Full access: read, write, delete and configuration |
| `subadmin` | Read and register only |

---

## 💱 Supported Currencies

- Venezuelan Bolívar (Bs)
- US Dollar / USDT (USD)
- Euro (EUR)
